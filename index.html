等距房间编辑 + 光影一体化 Demo

这版把房间编辑系统 + 多光源系统 + 人物代理体积阴影合并在一起。人物的阴影不再按单点脚底处理，而是按一个可调尺寸的代理长方体投影到地面，并随移动实时更新。

世界 / 网格

说明：这里修改的是整个世界的尺寸与颗粒度。第一版中，颗粒度只作用于“新世界/重置世界”，不会强行重采样当前已有场景。

Habbo 导入：严肃参考 Scuti 的组织方式——直接读取 SWF 里的 objectData / visualization / assets / bitmap tags，先按 room object 的 floor anchor 放到场景里，再按 direction + layer offsets 组装 2D 显示，并自动生成 proxy。当前版不再 flatten 成单张 body 图，而是按 direction + layer 单独绘制，包含 shadow/body 等层；并内置多种 Habbo SWF 样例，方便直接回归测试。

R：重置人物

        
U：显示/隐藏光轴
