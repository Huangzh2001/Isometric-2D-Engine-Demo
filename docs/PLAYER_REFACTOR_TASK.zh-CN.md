# Player 重构任务说明

## 目标
将人物运动与碰撞系统抽离到 `src/application/player/player.js`，在保持行为不变的前提下明确职责边界。

## 本轮范围
集中以下内容：
- 人物移动入口
- 输入到位移的转换
- 与世界边界和已放置 box 的碰撞
- 人物空间辅助函数

## 非目标
- 不重写遮挡/排序算法
- 不修改光影行为
- 不修改 placement 职责
- 不修改 scene / prefab 数据结构

## 验收
- 主程序可正常打开
- 人物移动仍能响应输入
- 世界边界碰撞正常
- 与已放置物体的碰撞正常
- placement 和 scene save/load 不受影响
