  • 右侧滚轮：切换当前基准层 L；图片模式下按住 Ctrl 再滚轮可缩放图片。
  • 显示规则：z = L 蓝色高亮，z > L 正常颜色，z < L 灰色半透明仅参考；矩形/体积工具在点下第一个起点后才会进入实时预览。
  • 单格选区：可选第 L 层空格，或选择已有 voxel 的顶面（只限 z >= L）。
  • 矩形选区：在第 L 层选一个二维矩形。
  • 体积选区：先选底面矩形，再上下移动鼠标定高度，最后点一下或按 Enter 确认选区。
  • 右侧只负责“选区”，真正生成/删除由左侧按钮执行；Esc 清空当前选区流程。
